GAT Sistema de Gestão de Ativos de TI

Resumo

Sistema Gestão de Ativos de TI (GAT) desenvolvido em typeScript com Node.js no backend e React no frontend, utilizando banco de dados SQLite. O sistema permite o cadastro, monitoramento e gerenciamento eficiente dos ativos de tecnologia da informação, facilitando o controle de emprestimos, manutenção e controle dos recursos de TI nas organizações gerando dashboard, termos e relatórios.

Competências Adquiridas
  • Desenvolvimento de lógica de negócios complexa no front-end e backend (CRUD completo).
  • Manipulação avançada do DOM para criar interfaces dinâmicas e interativas.
  • Controle de acesso dos técnicos e gerenciamento de recursos e ativos pelo gestor.
  • Criação de layout responsivo e amigável focando na experiência do usuário.
  • Disponibilidade de uso via rede intranet ou web
