Sistema de Registro para Triagem

Resumo do Projeto

Aplicativo simples em C# para registro de triagem dos chamados feito via telefone/chat web, com controle de anotações e script com opção para exportar registros.

Tecnologias Utilizadas
C# .NET Framework Windows Forms Visual Studio 2019
Competências Adquiridas
  • Desenvolvimento de lógica de negócios complexa em C#.
  • Implementação de interface gráfica com Windows Forms.
  • Utilização do Visual Studio 2019 para desenvolvimento e depuração.
  • Gerenciamento de dados e exportação de registros usando bibliotecas .NET.
