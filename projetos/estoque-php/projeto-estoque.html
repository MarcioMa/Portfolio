Sistema de Gestão de Estoque

Resumo

Sistema de Gestão de Estoque desenvolvido em PHP com MySQL, permitindo o cadastro, controle e gerenciamento eficiente dos produtos em estoque. O sistema oferece funcionalidades como cadastro de produtos, movimentações de entrada e saída, relatórios de estoque e controle de fornecedores.

Tecnologias Utilizadas
  • PHP
  • MySQL
  • HTML
  • CSS
  • Bootstrap
    • Competências Adquiridas
    • Desenvolvimento de lógica de negócios complexa no front-end e backend (CRUD completo).
    • Manipulação avançada do DOM para criar interfaces dinâmicas e interativas.
    • Variaveis de ambiente e configuração de banco de dados.
    • Controle de rotas e acesso ao sistema usando sessões e cookies.
    • Criação de páginas com HTML Semântico.
